## Documentation
Set alias to instance of Ringbuffer. When not set, Alias is is the same "RingBuffer.[name of type <T>]"

### Methods

```csharp
  IRingBuffer<T> AliasName(string value)
``` 

### Exception

 When alias is null or empty.

```csharp
  RingBufferFatalException
```
